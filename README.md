# Twins Journey Website

This repository contains a simple static website chronicling the growth of our
twin boys. It includes:

- A home page with navigation
- Individual posts for milestones like walking, running, and cycling
- A yearly photo gallery that loads image URLs from `data/gallery.json`

To view the site locally, open `index.html` in your browser. Add new images to
the gallery by updating `data/gallery.json` with additional years and URLs.
